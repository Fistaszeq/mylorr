Code Block to jest środowiskiem trochę już starym i sztywnym, stylów nie da się zmienić ale preferuję go dla osób które zaczynają swoją przygodę z programowaniem. Code Block jest też często używany na maturze z informatyki do C++/C.

Do instalacji oczywiście najpierw trzeba pobrać program, zalecam ze strony głównej Code Block. Link do strony Code Block.

Klikamy w "Donwload the binary relase ", szukamy zakładki Microsoft Windows.

Klikamy na wesje z mingw, najlepiej z setup'em. Przechodzimy proces instalacyjny jak noramlny program. Oczywiście pobieramy wersję z kompilatorem MinGW.

W przypadku jakich kolwiek błędówm, na tej stronie nie znajdziecie odpowiedzi narazie. Polecam poszukać na YouTube'ie wpisać kod błędu w przeglądarce czy coś podobnego. Googlujcie wszystko czego nie wicie albo wam nie działa to nic złego.
